# 한강 수질 오염 분석 | Hangang Water Quality Analysis

한강의 지점별 수질 오염도를 분석하고 시각화하는 데이터 분석 프로젝트입니다.

---

## 🎯 프로젝트 목적

한강의 수질 현황을 파악하고, 지점별/계절별 오염 특성을 이해하는 것을 목표로 합니다.

---

## 📊 분석 목표 (구체적 분석 항목)

1. **한강 지점별 수질 오염도 비교**
   - 5개 측정소의 BOD, COD, TP 등 주요 오염 지표 비교
   - 한국 환경기준(Ⅰ~Ⅲ등급) 대비 현황 파악

2. **상류→중류→하류 오염도 변화**
   - 상류(팔당댐2) → 지류(성내천) → 중류(보광, 노량진) → 하류(영등포)의 변화 경향 파악
     
3. **계절 및 월별 오염도 변화 추이**
   - 계절에 따른 수질 변화 패턴 분석

4. **Tableau 대시보드 시각화**
   - 분석 결과를 직관적으로 표현

## 📈 분석 결과

### #02 EDA - 박스플롯 분석
**분석 항목:** DO, BOD, COD, TP, TN (지점별 오염도 분포)

**결과:**
- 모든 지점에서 II등급 이내의 분포 확인
- 팔당댐2(상류)에서 중앙값이 가장 낮음
- 상류→하류로 갈수록 박스의 위치 상향 이동
- 모든 지표에서 이상치(outlier) 없음
- 환경기준 초과 지점 없음

---

### #03 지점별 평균값 비교 
 
**분석 항목:** 5개 지점의 평균값 시각화 (바차트)
 
**주요 결과:**
 
| 지표 | 특징 |
|------|------|
| **DO** | 전 지점 Ia등급으로 산소 포화도 우수 |
| **BOD** | 증가 추세 있으나 II등급 이내로 양호 |
| **COD** | 중류에서 소폭 상승 후 안정화 |
| **TP** | 상류→하류 약 1.35배 증가로 주의 필요 |
| **TN** | 뚜렷한 증가 추세 (기준 없음) |
