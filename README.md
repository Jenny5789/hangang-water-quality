# 한강 수질 오염 분석 | Hangang Water Quality Analysis

한강의 지점별 수질 오염도를 분석하고 시각화하는 데이터 분석 프로젝트입니다.

---

## 🎯 프로젝트 목적

한강의 수질 현황을 파악하고, 지점별/계절별 오염 특성을 이해하는 것을 목표로 합니다.

---

## 📊 분석 목표 (구체적 분석 항목)

1. **한강 지점별 수질 오염도 비교**
   - 5개 측정소의 BOD, COD, TP 등 주요 오염 지표 비교
   - 한국 환경기준(Ⅰ~Ⅲ등급) 대비 현황 파악

2. **상류→중류→하류 오염도 변화**
   - 상류(팔당댐2) → 지류(성내천) → 중류(보광, 노량진) → 하류(영등포)의 변화 경향 파악
     
3. **계절 및 월별 오염도 변화 추이**
   - 계절에 따른 수질 변화 패턴 분석

4. **Streamlit 대시보드 시각화**
   - 분석 결과를 인터랙티브하게 표현

## 📈 분석 결과
### ##1 데이터 준비
 
**데이터 로드 및 기초 정보:**
- 5개 지점 × 36개월 
- 결측치 제거 후 균등화
- 9개 컬럼: 측정소명, 년/월/일, DO, BOD, COD, TP, 수온, TN, SS

### #02 EDA - 박스플롯 분석
**분석 항목:** DO, BOD, COD, TP, TN (지점별 오염도 분포)

**결과:**
- 모든 지점에서 II등급 이내의 분포 확인
- 팔당댐2(상류)에서 중앙값이 가장 낮음
- 상류→하류로 갈수록 박스의 위치 상향 이동
- 모든 지표에서 이상치(outlier) 없음
- 환경기준 초과 지점 없음

---

### #03 지점별 평균값 비교 
 
**분석 항목:** 5개 지점의 평균값 시각화 (바차트)
 
**주요 결과:**
 
| 지표 | 특징 |
|------|------|
| **DO** | 전 지점 Ia등급으로 산소 포화도 우수 |
| **BOD** | 증가 추세 있으나 II등급 이내로 양호 |
| **COD** | 중류에서 소폭 상승 후 안정화 |
| **TP** | 상류→하류 약 1.35배 증가로 주의 필요 |
| **TN** | 뚜렷한 증가 추세 (기준 없음) |
 
---
 
### #04 시계열 분석
 
**분석 항목:** 월별(36개월), 계절별(4계절), 년도별(3년) 오염도 변화
 
**월별 변화 추이:**
- DO: 8~15 ㎎/L의 범위로 계절 변동성 명확
- BOD: 0.8~2.5 ㎎/L로 월별 변동 있음
- TP: 0.03~0.10 ㎎/L의 편차
**계절별 특성:**
 
| 계절 | DO | BOD | 특징 |
|:---:|:---:|:---:|------|
| **봄** | 11~12 | 1.3~1.5 | 중간 수질 |
| **여름** | 13~15 | 0.8~1.2 | 최고 수질 |
| **가을** | 10~11 | 1.5~1.8 | 악화 추세 |
| **겨울** | 5~8 | 2.0~2.5 | 최저 수질 |
 
**년도별 추세:**
- 2023년: DO 평균 9.7 ㎎/L
- 2024년: DO 평균 10.1 ㎎/L (소폭 개선)
- 2025년: DO 평균 10.1 ㎎/L (안정)
- 2026년: DO 평균 12.2 ㎎/L (뚜렷한 개선) ✅
  
**결론:**
- 계절별 변동이 자연적 패턴 (수온 변화)
- 최근 수질 개선 경향 명확
- 지류 관리 필수
