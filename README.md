# 한강 수질 오염 분석 | Hangang Water Quality Analysis

한강의 지점별 수질 오염도를 분석하고 시각화하는 데이터 분석 프로젝트입니다.

---

## 🎯 프로젝트 목적

한강의 수질 현황을 파악하고, 지점별/계절별 오염 특성을 이해하는 것을 목표로 합니다.

---

## 📊 분석 목표 (구체적 분석 항목)

1. **한강 지점별 수질 오염도 비교**
   - 5개 측정소의 BOD, COD, TP 등 주요 오염 지표 비교
   - 한국 환경기준(Ⅰ~Ⅲ등급) 대비 현황 파악

2. **상류→중류→하류 오염도 변화**
   - 팔당댐2(상류) → 노량진(중류) → 보광(하류)의 변화 경향 파악

3. **계절 및 월별 오염도 변화 추이**
   - 계절에 따른 수질 변화 패턴 분석

4. **Tableau 대시보드 시각화**
   - 분석 결과를 직관적으로 표현

## 📈 주요 분석 (박스플롯)

**분석 항목:** BOD, COD, TP (지점별 오염도 분포)

**결과:**
- 모든 지점이 한국 환경기준 Ⅱ~Ⅲ등급 수준
- 상류(팔당댐2)에서 하류(보광)로 갈수록 오염도 증가 경향
- TP(총인)가 가장 높은 오염 지표
